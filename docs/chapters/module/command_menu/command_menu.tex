\LevelOne{command\_menu}\label{ch:command_menu}

\LevelTwo{Purpose}
Define menu GUI to execute commands.


\LevelTwo{Command}
\fcmd{/command-menu}

\begin{note}{Use command bundle module to create the interface for normal users}
    The \cmd{/command-menu open} command is only for admin players.
    You need to use~\nameref{ch:command_bundle} module, to create a new command, for normal users.
\end{note}
