\LevelThree{mention}
\LevelFour{Purpose}
Insert the player name in the chat message, and the target player will be mentioned and sound notified.

\begin{note}{Problems with Styled Chat mod}
    The \ttt{chat.mention} module partial work with \ttt{Styled Chat} mod.
    You can use \ttt{chat.mention} module with \ttt{Styled Chat} mod, the target player still get mentioned, but the mention\_format will not be rendered.
\end{note}
