\LevelThree{replace}\label{ch:chat_replace}
\LevelFour{Purpose}
Allow you to define \ttt{regex} to replace \ttt{chat string} with a given \ttt{chat text}.

\begin{warn}{Didn't work with Styled Chat mod}
    The \ttt{chat.replace} module does work with \ttt{chat.style} module.
    But if you are using \ttt{Styled Chat} mod to handle the chat style, then this module will not work.
    It's because the \ttt{Styled Chat} mod hacks the Minecraft internal chat processing flow, and making the \ttt{chat.replace} module failed to work.
    However, since \ttt{Styled Chat} mod does support you to define custom \ttt{emotions}, which provides the same purpose as \ttt{chat.replace} module.
    \tbf{So, if you are using \ttt{Styled Chat} mod, you didn't need the \ttt{chat.replace} module, you can disable it!}
\end{warn}

\LevelFour{Example}
\begin{example}{Define a chat shortcut to evaluate a placeholder}
    \begin{jsoncode}
    {
        "regex": "(?<=^|\\s)uuid(?=\\s|$)",
        "replacement": "<green>my uuid is %player:uuid%</green>"
    }
    \end{jsoncode}
\end{example}
