\LevelZero{Command}\label{ch:command}

\LevelOne{Definition}
A \ttt{command} is a \ttt{chat string} used to perform a specific action/function.\\

\begin{tips}{List all commands.}
    Issue \cmd{/fuji} command, to list all the enabled modules.\\
    Issue \cmd{/fuji inspect fuji-commands}, to list all the provided commands by enabled modules.
\end{tips}
