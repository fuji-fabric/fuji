\LevelZero{Command}\label{ch:command}

\LevelOne{Definition}
A \ttt{command} is used in Minecraft to perform a specific action/function.\\
Fuji provides lots of commands for various purpose.\\
Commands are registered by a module.\\
By default, all modules are disabled, so you have to enable modules in \mainconfig file, and re-start your server to apply the changes.

\LevelOne{What are the commands provided by this mod?}

Issue \cmd{/fuji} command, to list all the enabled modules.\\
Issue \cmd{/fuji inspect fuji-commands}, to list all the registered commands by enabled modules.
